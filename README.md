# myStl

Проект собирался в Visual studio 2019 x64.

---
### Vector
Реализация структуры данных vector. Вариант с эффективным выделением память(используется ::operator new, а не обычный new).

### List
Реализация структуры данных list. 

### Binary search tree (BSTree)
Реализация структуры данных бинарное дерево поиска.
</br>

_Во всех структурах данных используются самописные итераторы._
